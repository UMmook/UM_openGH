태그명 tab

태그 여러개: 태그 * 갯수

포합관계 만들기 : 태그 > 태그 > 태그

조합 예제 : div>ul*2>li*2

조합 예제 : div>(ol>li*2)+(ul>li*2)

컨텐츠 표현 : div{div 컨텐츠}

컨텐츠 반복 표현 : div*3{내용}

컨텐츠 반복 표현 $는 숫자 : div*3{내용$}

id를 사용하는 태그 : div#id1

활용 : div#id$*3

활용 : div#id$*3{내용$}

class를 사용하는 태그 : div.c$*3{내용$}
